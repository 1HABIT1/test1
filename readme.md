 ## Git安装

 ## 初始化Git仓储/(仓库)
 - 这个仓库会存放，git对我们项目代码进行备份的文件
 - 在项目右键打开 git bash
 - 命令：'git init'

 ## 自报家门
 - 命令：
    + 配置用户名：'git config --global user.name "liuhua"'
    + 配置邮箱：'git config -- global user.email "xxx@xxx.com'

 ## 把代码存储到.git仓库中
 - 1. 把代码放到仓储的门口
    + 'git add ./readme.md'
 - 2. 把仓储门口的代码放到里面的房间里去
    + 'git commit -m "这是对这次添加东西的说明"'
